Smart Expense Tracker - Backend

Managing personal finances can be difficult, especially when relying on manual methods like spreadsheets or scattered notes. The Smart Expense Tracker Backend provides a robust foundation for logging, categorizing, and analyzing expenses, while also handling user authentication and data storage. This backend service is built with Flask (Python), follows Object-Oriented Programming (OOP) principles, and connects with a SQLite database to manage users and expenses.

Features

User Management: Registration, login, authentication, and profile management.
Expense Tracking: Add, edit, delete, and categorize daily expenses.
Data Storage: Persistent storage with SQLite.
RESTful API: Clean and modular API endpoints for frontend integration.

Tech Stack

Backend Framework: Flask (Python)
Database: SQLite
